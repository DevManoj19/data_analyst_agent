# Data_analyst_agent

This notebook demonstrates how to:

1.  Install necessary Python libraries.
2.  Set up authentication for the Together.ai API.
3.  Upload and read various file types (.csv, .xlsx, .txt, .pdf, .docx, .png, .jpg, .jpeg).
4.  Analyze uploaded data if it's a pandas DataFrame.
5.  Use the Together.ai API to ask questions using the Llama-4 Maverick model.
6.  Optionally visualize data (specifically, generate a correlation heatmap for DataFrames).

## Installation

The required Python libraries are installed at the beginning of the notebook using `pip`.
